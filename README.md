# CSharp_BoardGames
C# Console Application for various Board Games

University Assignment developed using C# .NET

Supported Games
  1. TicTacToe
  2. Connect4
  3. Reversi

Supported Modes
  1. Human vs Human
  2. Human Vs AI
  
Extra Functions
  - Saving and Loading
  - Undo and Redo
  - Interactive Command System
